# Vanilla.ts

Web UIs with pure TypeScript.

Opinionated, highly experimental, unfinished. Don't use.

*Components*, requires [vanilla-ts-core](https://github.com/mn4367/vanilla-ts-core) and
[vanilla-ts-dom](https://github.com/mn4367/vanilla-ts-dom).

Currrently a directory sructure like this is required:

```
<parent-dir>
  |- vanilla-ts-core
  |- vanilla-ts-dom
  |- vanilla-ts-components
```
